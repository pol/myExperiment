Quick notes for installation from the repository:

  (1)  copy config/database.yml.pre to config/database.yml

  (2) copy config/default_settings.yml.pre to config/default_settings.yml.pre

These files contain passwords for MySQL and SMTP respectively.
  (3) update the contents of each of the files to match your desired setup

For a more detailed guide of how to install MyExperiment, including all the Ruby-on-Rails support, please see developers.txt.
